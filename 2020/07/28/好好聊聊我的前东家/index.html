好好聊聊我的前东家

发表于 2020-07-28 分类于 忙忙碌碌皆为宿命 阅读次数： Valine：
本文字数： 2.8k 阅读时长 ≈ 3 分钟

　　静下心来，好好写写我的前东家吧
　　虽然已经离开了一段时间，虽然离开的时候不是很愉快，虽然现在还和前东家没有扯清经济帐（无经济纠纷，单纯个人押金与报销清算缓慢）。但是仍然要说汉得是一家挺不错的公司，记住这里所说的汉得是真正的汉得，并不包含某逸，某晶等子公司，各位同学不要一概而论。至于我认为汉得不错，我大致可以列举为以下三个理由吧。
1.汉得给予人的空间足够
咨询公司已经没有十年前的高端人设，参考《我的前半生》电视剧中靳东饰演的贺涵这样的咨询顾问早就成为了历史，收入较十年前没变化，干得活更加苦逼，你没时间去耍流氓和潜规则了，当然时间管理大师除外，汉得这边也爆出了几个。
咨询顾问这个对于经验的要求其实是特别高的，一般从最基本的业务了解、咨询方法论、案例实践等等，能担任到项目经理一般至少都是3-5年时间，但是汉得这一点不一样，因为人员流动特别大，所以很多经验不足的顾问被迫顶上来做项目经理，这样极其容易让项目出现比较大的问题，所以很多客户反馈说需要经常给项目经理讲业务，梳理业务。也是因为这个原因导致了顾问不了解需求，不了解市场，更不了解变化趋势，导致客户不满意，项目经理心力憔悴，项目成员不知道做什么。这个应该是现在很多很多很多项目的通病。
以上看似是一个全盘皆输的现象，但是如果我们想一下，你是一个特别有能力的人，不甘于熬资历、等考核，这是不是就是一个很好的展现。如果你能力突出，能够挽狂澜于既倒，扶大厦于将倾，汉得绝对能让你凸显出来。我曾经遇到一个技术顾问，实习经历特别不错，技术能力十分牛逼，领导直接就说，两个项目后你就可以做技术经理。而且相应的工资待遇，都有不错的照顾。自夸一点，我觉得我本人也是，虽然没有拿到汉得的股票啊，优秀员工之类，但是每次升级考核都有不错的分数。每次级别晋升考核也都有薪酬的提升。这一点是让很多有能力的人感到快速的满足。但是至于比较平庸甚至比较差的，那么肯定也都会有相应的那个啥。
所以说我认为，汉得是一家能够给予人快速上升通道的公司。新人有充分的空间，能够快速凸显自己的能力和得到锻炼的机会。同时也会承认会有一部分人没有太强的能力也会上去，这一点呢，在哪都避免不了。在汉得存在26岁的二级事业部经理（H11-H12，相当于四大原manager）这一点在其他咨询公司可能比较难吧（我才疏学浅，说错了不要打我），当然也有一个混了十几年还在做一个普通的咨询顾问。人各有命，没招。
2.汉得是一家能让你生存的公司
当然我说的可能是以前，不一定是现在。我当时去汉得时候，汉得还能够帮忙解决顾问的住宿问题，无论是在项目上还是在园区，只要你想住，肯定有免费的地方让你住。并且在项目上还基本上都是单间，这对于一个在外工作的人来说是何等的幸福，在汉得三年时间，从来没有因为住宿发过愁，出差住汉庭、格林、如家，在项目上住项目租的房子（并且一般不会离上班地方太远）这一点我觉得特别好，你的工资和补贴加起来基本上让你吃吃喝喝就行了。按照我当时毕业生进来应该在5.5k—7k之间，然后每天补贴有80，周末可以在外也可以补。然后还有工作日餐补。这样一个月基本上接近1w的水平。所以说这样活着还是不错的。
当然在2019年和2020年貌似出现了转变，看其他知乎的帖子，已经开始说转正工资只给到了4k，然后补贴也变成了只有工作日，取消了餐补。2019届的新人收入水平瞬间下降到了6k左右，这个水平对于汉得校招院校定位是不符的。这个待遇还是辜负了这群985和211的毕业生。
关于汉得的FB政策，本质是为了让大家出差的时候经常照顾家里，所以原来每双周给大家一张往返机票，让大家回家看看，如果不使用可以折合成800元人民币，这样基本上每个人一个月又多了1600元补贴。但是据说某些子公司不是这样的，前面提到的某晶，貌似一个月只有一次。到了H9以后才能每个月两次。至于现在是什么政策，我就不清楚了，毕竟时代在发展，公司的政策也在变化。
综上按照我还在的时候那个水平，一个月的收入基本上为：6k+2.4k+0.3k+1.6k=10.3k，基本上到手在9K+,这些相当于有12k工资的水平，并且还不用担心住宿，所以我说，汉得是一家能让你生存的公司，至于你拿这到手的9K活成什么样我就不知道了，有的三年攒了40w，有的三年啥也不落。我很不幸就是啥都不落的那种。
3.汉得是一家让你越来越心累的公司
其实一开始，我本来想写失望的，但是想了想还是用心累吧。因为不一定是公司的问题，可能是自身的问题。
在汉得呆了三年，三年来不停的在接触新客户、新同事（因为熟悉的同事在不停的离职）、然后每个项目可能会搭档不同的伙伴，虽然大家相处的都还不错，但是都是有一个缓慢认识的过程，有的时候好不容易磨合的差不多了，有了默契，结果下个项目就分开了。我在汉得的三年里，不断传授项目管理知识、业务知识、与人磨合，总体来说还是有些心累的。这是其中一点。
那么第二点就是长期出差带来的奔波感，在汉得成出差时间长短不一，长的一年两年，短的1天两天。反正不是在出差就是在出差的路上。你的家当永远是那个行李箱。刚毕业的时候觉得这样很新鲜，激情满满。可是当你坐了几百次高铁，飞了几十次夜航班之后，你已经不知道自己在哪并且想去哪了，连一个候鸟都不如。每次换项目总有一堆东西要扔，到了新项目又有一堆东西要买，当你想好好生活的时候，发现行李箱根本装不下。这种心累是其二。
在2018年年会上，有一个短视频戳中了无数Hander的泪点，是一位资深顾问讲的他的故事；他家在西安，儿子三四岁，因为长期在外做项目疏于对儿子的照顾，每双周的FB他都会飞回去，然后儿子看到行李箱就知道他回来了，周一早上儿子起床看到他行李箱没了就知道他又走了，所以他儿子有时候会说他爸爸就是一个行李箱，行李箱在爸爸就在，行李箱不在了爸爸就不在。后来这个顾问听说了以后，买了个新的行李箱，把旧的放在家里。这样能让他儿子感觉到爸爸一直在。听完这个故事，我们那一桌的老顾问泪腺就像破了一样，根本拦不住。后来一个在2019年转岗回家了，另外一个2020年离职了，都回到各自的城市。也许家庭都是我们最后的归宿，但是顾问是没有家的。这就是心累的第三点。
2020年我也因为种种因素离职，在汉得做了三年，公司也在大发展，铁打的营盘流水的兵有人走有人来。在汉得三年，收获失去了很多，得到了也很多。现在换到一个省会城市，上个轻松的班。每天能按时上班，按时下班。和心爱的人一起生活，买菜，做饭。
也许汉得很好，但是作为一个人，我可能过分的想要生活。
谨祝各位Hander前程似锦。
PS:其实网上看了很多关于汉得的帖子，作为一名渣渣的国内顾问。没有做过global的项目，没有出过大陆。所以什么在荷兰看风车，在巴黎爬铁塔的，我并没有感受过，可能是另外一种滋味。现在也没有股票了，走就走了，一身轻松。另外关于各种渣男什么的事情，也只能说一句鸟儿大了，什么林子都有。
